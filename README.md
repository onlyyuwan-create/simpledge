# 📱 度支简账 (Simpledge)

一款完全本地运行的记账软件，像"钱迹"一样好用，数据全部存在你的手机里。

## ✨ 功能

- ✅ **记一笔**：记录支出/收入，支持分类、日期、备注
- ✅ **账单列表**：按月查看，可按类型筛选
- ✅ **统计图表**：饼图看分布，趋势图看变化
- ✅ **分类管理**：自定义支出/收入分类
- ✅ **数据导出**：支持 JSON 备份 / CSV 表格
- ✅ **数据导入**：从 JSON 备份恢复
- ✅ **离线可用**：首次加载后无需网络
- ✅ **本地存储**：所有数据在手机，不上传云端
- ✅ **PWA 支持**：可添加至桌面，像原生 App 一样使用

## 📲 安装方法

### 方法一：使用 HTTP 服务器（推荐）

1. 将 `account-book` 文件夹传到手机上（USB 或微信文件传输助手）

2. 在手机上安装 Termux（可从酷安/官网下载），然后运行：
   ```bash
   cd /sdcard/Download/account-book  # 根据实际路径调整
   python -m http.server 8080
   ```

3. 打开手机 Chrome 浏览器，访问：`http://localhost:8080`

4. 当页面加载后，Chrome 地址栏右侧会出现「安装」图标（或菜单里选「添加到主屏幕」）

5. 点击安装，App 图标会出现在桌面，以后点图标就能用

### 方法二：使用 HTTP Server App

1. 在华为应用市场或酷安搜索 "HTTP Server" 或 "Simple HTTP Server"
2. 安装后，选择 `account-book` 文件夹作为根目录
3. 启动服务器，记下显示的地址（如 `http://192.168.1.x:8080`）
4. 打开 Chrome 访问该地址
5. 添加到主屏幕

### 方法三：使用 Python 临时启动

如果你在电脑上：
```bash
cd account-book
python3 -m http.server 8080
# 然后在同一 Wi-Fi 下的手机上访问 http://电脑IP:8080
```

## 🎯 使用指南

### 记录一笔账
1. 首页点击右下角 **+** 按钮
2. 选择**支出**或**收入**
3. 输入金额（点击数字区域直接键盘输入）
4. 选择分类（餐饮、交通、购物等）
5. 可选：修改日期、添加备注
6. 点击 **保存**

### 查看账单
- 底部切换到「账单」标签
- 左右箭头切换月份
- 点击「全部/支出/收入」筛选

### 查看统计
- 底部切换到「统计」标签
- 饼图查看分类占比，趋势图查看每日变化
- 左右箭头切换月份

### 备份数据
- 底部「设置」→「导出 JSON」导出完整备份
- 「设置」→「导入数据」恢复备份

## 🔒 隐私声明

**你的数据 100% 保存在你的手机本地。**
- 使用浏览器内置 IndexedDB 数据库存储
- 没有任何云端服务器
- 不会上传任何数据
- 不需要注册账号
- 卸载前记得先导出备份！

## 🚀 GitHub Pages 部署（在线访问方式）

你也可以通过 GitHub Pages 直接在线访问，无需本地服务器：

1. 在 GitHub 上创建仓库（如 `account-book`）
2. 把项目文件推送到仓库
3. 仓库 Settings → Pages → 选择 `main` 分支，根目录
4. 等待几分钟，访问 `https://你的用户名.github.io/account-book/`
5. 用手机 Chrome 打开这个网址 → 添加到主屏幕 ✅

> 💡 这种方式更方便：以后我更新代码，你 `git pull` 拉取，Pages 自动部署，手机上刷新就是新版！

## ⚙️ 技术栈

- **PWA** (Progressive Web App) — 可安装、离线运行
- **IndexedDB** (Dexie.js) — 本地数据库
- **Chart.js** — 图表渲染
- **Vanilla JS** — 无框架依赖
